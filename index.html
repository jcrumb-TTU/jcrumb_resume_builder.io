About This Project

This application was developed as part of CSC 3100 at Tennessee Technological University during the Spring 2026 semester. It was built using AI-assisted development tools under the direct oversight and authorship of the student developer. All AI-generated code was reviewed, tested, and approved by the author prior to being committed to the project repository. Development operated within the coding conventions defined in the course AGENT.md file.

This project was developed within the academic parameters of CSC 3100 at Tennessee Technological University. The AGENT.md conventions file governing the project defined coding standards including Hungarian notation with camelCase, async/await patterns, RESTful API design, input validation using the blnError and strMessage pattern, WCAG 2.1 accessibility standards, Bootstrap 5 utility class usage, and local dependency management rules. How to Run This Application
  1. Prerequisites — Node.js v18 or higher must be installed. Download from https://nodejs.org.
  2. Install dependencies — Run npm install in the project root. This installs all packages and triggers postinstall.js which copies Bootstrap, SweetAlert2, FontAwesome, and jsPDF into their local directories automatically.
  3. Configure environment — Copy .env.example to .env. Open the new .env file and replace your_gemini_api_key_here with a valid Gemini API key. A free key can be obtained at https://aistudio.google.com/apikey — select the Gemini 2.5 Flash model.
  4. Run the application — Run npm start in the project root.
  5. Run in development mode — Run npm run dev to launch with DevTools open for debugging.
  6. Notes — resume.db is created automatically on first run. The .env file must never be committed to source control. If port 3000 is already in use change HTTP_PORT in .env.
Transferring to Another Machine

The project can be zipped and run on a separate device. The zip must exclude the .git folder and the node_modules folder. On the receiving machine follow these steps:

  1. Install Node.js from https://nodejs.org
  2. Unzip the project folder
  3. Open a terminal in the project folder and run npm install
  4. Create a .env file in the project root and add a valid GEMINI_API_KEY — see .env.example for the required format
  5. Run npm start

The resume.db file can optionally be included in the zip if you wish to transfer existing resume data to the new machine. If it is not included a fresh empty database will be created automatically on first run.
